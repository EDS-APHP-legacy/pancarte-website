Visualization

This module is dedicated to the development of a WebApp (back-end + front-end) for the two following needs:

  • History-exploration and Annotation
  • Real-time dashboard

Back-end source code: github.com/EDS-APHP/pancarte-backend
Front-end source code: github.com/EDS-APHP/pancarte-front

Prototypes

Real-time dashboard of all rooms

This page lists all available rooms, and for each "card" you will see the patient's name, vital signals available, its main diagnosis...
When you click on such a "card", you are being redirected to the "Real-time streaming of a room" page.

Real-time streaming of a room

This page shows in real-time:

  • Raw waveforms and numerics from monitors (expected delay of 5s max.)
  • Derived data computed via Spark or machine learning applications
  • Alerts on thresholds and other derived data

Historical data search

Coming soon.

This page allows to browse the history of all recorded data (in the pancarte-db) based on filters. When you selected a record, you can annotate it using the "Annotation of historical data" page.

Annotation of historical data

Coming soon.

This page allows to:

  • Zoom in/out, scroll, resize waveforms/numerics
  • Create new annotations (timstamp or timerange based)
  • Explore existing annotations, update them...
